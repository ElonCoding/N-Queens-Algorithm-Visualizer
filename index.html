How the Algorithm Works

Step 1: Place Queen

Try placing a queen in the current row, one column at a time.

Step 2: Check Safety

Check if the queen can attack any previously placed queens.

Step 3: Recurse or Backtrack

If safe, move to the next row. If not, try the next column.

Step 4: Solution Found

When all queens are placed safely, we've found a solution!
